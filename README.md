#### 一、配置[Github上的SSH](https://github.com/settings/keys)

```
1、验证是否有ssh keys
   ls -al ~/.ssh
2、如果有.pub结尾的文件直接打开
   cat ~/.ssh/id_rsa.pub
3、如果没有新建ssh keys 回车生成公私钥
   ssh-keygen -t rsa -b 4096 -C "自己邮箱号"
4、github进行配置ssh   
5、验证是否和github链接
   ssh -T git@github.com   
   
```
